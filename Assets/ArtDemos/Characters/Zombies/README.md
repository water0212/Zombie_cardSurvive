# Zombie Character Assets

Simple 2D zombie character asset for the card survival prototype.

## Files

- `transparent/zombie_simple_2d.png`: final transparent PNG.
- `sources_chromakey/zombie_simple_2d_source.png`: original generated chroma-key source.
- `zombie_simple_2d_preview.png`: checkerboard preview for quick review.

## Style

- Simple 2D game sprite illustration.
- Not photorealistic.
- Low-to-moderate detail.
- No blood, gore, UI, card frame, or text.
