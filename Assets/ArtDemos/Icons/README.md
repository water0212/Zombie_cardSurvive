# UI Icons

Generated UI icon assets for the zombie survival card demo.

## Transparent PNGs

- `transparent/icon_food_can.png`: food icon, represented by a canned food item.
- `transparent/icon_resource_parts.png`: resource icon, represented by steel mechanical parts.
- `transparent/icon_combat_axe.png`: combat power icon, represented by an axe.

## Sources

- `sources_chromakey/`: original generated images on a chroma-key background.
- `icon_contact_sheet.png`: preview sheet on a checkerboard background.
